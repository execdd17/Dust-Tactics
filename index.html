Description

My evolving implementation of the game "Dust Tactics" created by Fantasy Flight Games [http://fantasyflightgames.com]

I am in no way affiliated with Fantasy Flight Games, but if this program seems interesting, I encourage you to buy the actual game!

Requirements

Ruby >= 1.9.3

Status

Version 1.0 is in development. Currently, The game can not be played from a central UI and a few remaining components must be designed and implemented.
